Milestone Two

My website will be one page, and will consist of the following sections:


Welcome banner / menu navbar:

Welcome banner will include a navigation bar which will list the different types of photography that Cindy offers. There will be six different types: portraits, drone, night sky, sports, landscapes, and creative. User will be able to tap a genre and page will automatically scroll to that genre where they will see the options menu again in order to jump to different photo types.

Contact button below navbar:

Here the user will be able to tap on "Contact Cindy" which will provide contact via email.

Photography gallery:

The website will serve as a gallery for some of Cindy's photography, showcasing each genre and it will smoothly scroll through all photos or user may choose to use the selection navigation bar to jump around between genres.
